# Финансовый анализатор транзакций

Проект для чтения и обработки финансовых операций из CSV и Excel файлов, расположенных на Рабочем столе.

## 📋 Описание проекта

Проект предоставляет инструменты для:
- Чтения финансовых транзакций из CSV-файлов
- Чтения финансовых транзакций из Excel-файлов (xlsx, xls)
- Автоматического определения формата файла
- Обработки числовых данных (автоматическое преобразование int/float)
- Обработки пустых значений (NaN → None)
- Модульного тестирования всех функций

- pip (менеджер пакетов Python)
