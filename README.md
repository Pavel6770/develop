# Виджет банковских операций
## Описание проекта
Виджет для личного кабинета клиента банка, отображающий несколько последних успешных операций. Разработан для IT-отдела крупного банка с акцентом на безопасность и удобство использования.

## Основные функции
### 🏦 Маскирование данных
Безопасное отображение конфиденциальной информации клиента.

get_mask_card_number(card_number: int) -> str
Маскирует номер банковской карты.

Вход: номер карты (число)

Выход: замаскированный номер в формате "XXXX XX** **** XXXX"

Пример: 1234567890123456 → 1234 56** **** 3456

get_mask_account(account_number: int) -> str
Маскирует номер банковского счета.

Вход: номер счета (число)

Выход: замаскированный номер в формате "**XXXX"

Пример: 12345678901234567890 → **7890

mask_account_card(card_info: str) -> str
Универсальная функция для маскировки карт и счетов.

Вход: строка с типом и номером карты/счета

Выход: строка с замаскированным номером

Примеры:

"Visa Platinum 1234567890123456" → "Visa Platinum 1234 56** **** 3456"

"Счет 12345678901234567890" → "Счет **7890"

### 📅 Работа с датами
get_date(date_str: str) -> str
Преобразует дату из ISO формата в удобный для чтения формат.

Вход: дата в формате ISO (YYYY-MM-DDThh:mm:ss.ms)

Выход: дата в формате ДД.ММ.ГГГГ

Пример: "2024-03-15T10:30:45.123456" → "15.03.2024"

### 📊 Обработка операций
filter_by_state(operations: list, state: str = 'EXECUTED') -> list
Фильтрует список операций по статусу.

Вход: список операций, статус для фильтрации

Выход: новый список с отфильтрованными операциями

Пример: фильтрация только успешных (EXECUTED) операций

sort_by_date(operations: list, reverse: bool = True) -> list
Сортирует операции по дате.

Вход: список операций, порядок сортировки

Выход: новый отсортированный список

Пример: сортировка от новых к старым (по умолчанию)

### Пример использования
python
from src.widget import mask_account_card, get_date
from src.processing import filter_by_state, sort_by_date

###### Получение списка операций (пример данных)
operations = [
    {
        'id': 1,
        'state': 'EXECUTED',
        'date': '2024-03-15T10:30:45.123456',
        'card': 'Visa Platinum 1234567890123456',
        'amount': '1000.00',
        'currency': 'RUB'
    },
    {
        'id': 2,
        'state': 'EXECUTED',
        'date': '2024-03-14T09:20:30.123456',
        'card': 'Счет 12345678901234567890',
        'amount': '5000.00',
        'currency': 'USD'
    }
]

###### Фильтрация успешных операций
successful_ops = filter_by_state(operations)

###### Сортировка по дате (сначала новые)
latest_ops = sort_by_date(successful_ops)

###### Отображение последних 5 операций
for op in latest_ops[:5]:
    # Маскирование карты/счета
    masked_info = mask_account_card(op['card'])
    # Форматирование даты
    formatted_date = get_date(op['date'])
    
    print(f"{formatted_date} {masked_info}")
    print(f"Сумма: {op['amount']} {op['currency']}\n")
Результат:

Сумма: 1000.00 RUB

14.03.2024 Счет **7890
Сумма: 5000.00 USD

## Безопасность
### Принципы маскировки
Номера карт: показываются первые 6 и последние 4 цифры

Номера счетов: показываются только последние 4 цифры

Конфиденциальность: полные номера нигде не сохраняются

### Рекомендации по использованию
Всегда используйте функции маскирования перед отображением данных

Не храните немаскированные номера в логах

Регулярно обновляйте зависимости для безопасности
